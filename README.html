Panel Structure Tools
Plugin User Guide

Welcome to the user guide for Panel Structure Tools. This minimalist tool is designed to automate Panel extraction, Detail positioning, smart labeling, and two-way data synchronization (BIM) directly with Vault Excel spreadsheets.

"Single Source of Truth" Rule: For geometric parameters (Area, Coordinates, Detail Quantity), the CAD Drawing is the Absolute Truth. Do not manually edit Panel/Detail data in Excel. If changes occur, modify the CAD drawing and Push the data back to Excel.

1. GLOBAL VAULT EXCEL SYNC

This area manages the data flow between the CAD drawing and the Excel file (Bill of Materials & Revision History) located on the server/Vault.

  • [...]: Click this button to browse to the Excel file for synchronization. You can select an existing file or type a new filename for the Tool to create automatically.
  • Sync Sheet & Revision (Tab 1 & 2): Synchronizes data for drawing frames (A1) and Revision History.
  • Sync Panel Engineering (Tab 3): Pushes Geometric data (Panel, Area, Detail, Qty) from CAD's RAM to the Excel spreadsheet.
  • [Pull Data] Button: Pulls data from Excel to CAD. This feature automatically reads the revision history (Tab 2) and creates/updates Amendment Blocks stamped on the CAD drawing.
  • [Push Data] Button: Pushes data from CAD to Excel based on the selected Checkboxes. Data in Tab 1 & 2 will be Safely Merged, while data in Tab 3 will be completely overwritten with the latest data from CAD.

2. PANEL AUTOMATION

This area automates the extraction of ship panels and the layout of Lifting Lugs and Guides.

  • Deck No: Enter the Deck number (e.g., type 6, the Tool will auto-generate Panel names like Deck 6 - 601C, 601P...).
  • Guiding Mode: Select the guide bracket type. Guide 3 is a smart 3-prong clamp, Guide 2 is a standard perpendicular corner bracket.
  • [Scan & Name Panels] Button:
    1. Click to select the Centerline (From Aft to Forward).
    2. Window Select the entire deck polygon area.
    3. The Tool will automatically filter, close gaps, assign names (C/P/S), distribute Balloons, attach Guides, and load the absolute precise polygon boundaries into RAM.
  • [Add Point (Manual)] Button: Add manual operation points. Click the button and look at the CAD Command Line to choose:
    • [Balloon]: Click to place additional manual lifting lugs.
    • [Guide2 / Guide3]: Click to attach additional guide brackets manually.
    • [UpdateSymbol]: Batch select existing Balloons to change their force symbols (Batch change to T1, T2, or T3).

3. PLAN VIEW TOOLS

Plan view processing tools, integrated with Artificial Intelligence for smart labeling and an Audit Interface.

  • [Open Panel Data] Button: Opens the Modeless Dashboard. Here, click the Scan Drawing button and select the Detail Block clusters. The X-Ray and Polygon Intersection algorithms will run in the background to accurately count the number of Details belonging to each Panel.
  • [Generate Labels] Button: Select Detail Blocks (or Master Block). The AI (Spatial Hashing algorithm) will automatically generate "Detail X" text and attach it to one of the 4 corners of the Detail frame to minimize collision (overlapping lines) with surrounding objects. Defaults to prioritizing the vertical CenterX axis.
  • [Update by Proximity] Button: Automatically updates the content of old Texts on the drawing to Detail names based on physical proximity (The Text closest to a Detail will adopt that Detail's name).

4. DETAIL VIEW TOOLS

System for managing Shop Drawings (A1 Frame) and Revision History.

  • Global Revision Bump: Check this box if you want to upgrade the Revision (e.g., from REV.A to REV.B) for ALL drawings currently in the list simultaneously.
  • [Sync & Update Sheet] Button: Scans A1 Frame Blocks on the drawing. The Tool will gather the entire drawing list into the DataGrid below. It also supports Auto-Heal (automatically restoring deleted Texts) when pulling data from Excel.
  • [Create Rev Block] Button: Select one (or multiple) rows in the DataGrid, enter Revision content. Click this button, and the Tool will automatically stack the Revision History Blocks neatly above the Title Block without overlapping.
  • DataGrid (Sheets & Revs): Central control panel. You can Click to select and automatically Pan/Zoom the drawing to the corresponding A1 frame on the CAD screen.

Appendix: Force Symbols System

The Tool uses standard White Solid Hatch (prints as solid Black) to represent force application points:

Symbol Type Shape Meaning (Standard)
T1 ▲ Triangle Vertical force (Used for Deck, Stowed & Lifting positions). This is the common type and assigned by default.
T2 ■ Square Vertical force (Used for Deck & Stowed positions).
T3 ● Circle Vertical force (Used only for Deck position).

Panel Structure Tools
Hướng dẫn sử dụng Plugin

Chào mừng bạn đến với tài liệu hướng dẫn sử dụng Panel Structure Tools. Tool được thiết kế tối giản nhằm tự động hóa quy trình bóc tách Panel, định vị Detail, dán nhãn thông minh và đồng bộ dữ liệu hai chiều (BIM) trực tiếp với bảng tính Excel Vault.

Quy tắc "Single Source of Truth": Đối với thông số hình học (Diện tích, Tọa độ, Số lượng Detail), Bản vẽ CAD là Chân lý. Không sửa thủ công dữ liệu Panel/Detail trên Excel. Nếu có thay đổi, hãy sửa trên CAD và đẩy (Push) lại lên Excel.

1. GLOBAL VAULT EXCEL SYNC

Khu vực này quản lý luồng dữ liệu giao tiếp giữa bản vẽ CAD và file Excel (Bill of Materials & Revision History) đặt trên hệ thống máy chủ/Vault.

  • [...]: Nhấn vào nút này để trỏ đường dẫn tới file Excel cần đồng bộ. Bạn có thể chọn file có sẵn hoặc gõ tên file mới để Tool tự tạo.
  • Sync Sheet & Revision (Tab 1 & 2): Cho phép đồng bộ dữ liệu quản lý Khung bản vẽ (A1) và Lịch sử sửa đổi (Revision).
  • Sync Panel Engineering (Tab 3): Cho phép đẩy dữ liệu Hình học (Panel, Area, Detail, Qty) từ bộ nhớ RAM của CAD ra bảng tính Excel.
  • Nút [Pull Data]: Kéo dữ liệu từ Excel về CAD. Tính năng này sẽ tự động đọc lịch sử sửa đổi (Tab 2) và tự động tạo/cập nhật các Block Amendment dập lên bản vẽ CAD.
  • Nút [Push Data]: Đẩy dữ liệu từ CAD lên Excel dựa theo các Checkbox bạn đã chọn. Dữ liệu Tab 1 & 2 sẽ được Merge (ghép nối an toàn), trong khi dữ liệu Tab 3 sẽ được ghi đè hoàn toàn bằng dữ liệu mới nhất từ CAD.

2. PANEL AUTOMATION

Khu vực tự động hóa bóc tách khoang tàu (Panel) và bố trí điểm nâng hạ (Lifting Lugs) cùng cữ dẫn hướng (Guides).

  • Deck No: Điền số thứ tự của Boong (VD: gõ 6, Tool sẽ tự sinh tên Panel là Deck 6 - 601C, 601P...).
  • Guiding Mode: Chọn kiểu ngàm dẫn hướng. Guide 3 là ngàm kẹp 3 râu thông minh, Guide 2 là ke góc vuông góc thông thường.
  • Nút [Scan & Name Panels]:
    1. Click chọn đường Centerline (Từ Đuôi tàu hướng về Mũi tàu).
    2. Quét chuột (Window Select) toàn bộ khu vực đa giác mặt boong.
    3. Tool sẽ tự động lọc, vá nét, đánh tên (C/P/S), rải Balloon, dán Guide và nạp bộ khung đa giác chính xác tuyệt đối vào RAM.
  • Nút [Add Point (Manual)]: Thêm các điểm thao tác thủ công. Bấm nút và nhìn xuống Command Line của CAD để chọn:
    • [Balloon]: Click để rải thêm điểm nâng hạ thủ công.
    • [Guide2 / Guide3]: Click để gắn thêm cữ dẫn hướng bằng tay.
    • [UpdateSymbol]: Quét hàng loạt các Balloon có sẵn để đổi ký hiệu lực (Đổi hàng loạt sang T1, T2, hoặc T3).

3. PLAN VIEW TOOLS

Các công cụ xử lý trên mặt bằng (Plan View), tích hợp Trí tuệ Nhân tạo để dán nhãn (Labeling) và Giao diện Kiểm toán (Audit).

  • Nút [Open Panel Data]: Mở cửa sổ Dashboard (Modeless). Tại đây, bấm nút Scan Drawing và quét các cụm Detail Block. Thuật toán X-Ray và Giao cắt Đa Giác (Polygon Intersection) sẽ chạy ngầm để đếm chính xác số lượng Detail thuộc về từng Panel.
  • Nút [Generate Labels]: Quét các Detail Block (Hoặc Master Block). AI (Thuật toán Spatial Hashing) sẽ tự động tạo chữ "Detail X" và gắn vào 1 trong 4 góc của khung Detail sao cho ít bị va chạm (đè nét) với các nét vẽ xung quanh nhất. Mặc định ưu tiên trục dọc CenterX.
  • Nút [Update by Proximity]: Tự động sửa nội dung của các Text cũ trên bản vẽ thành tên Detail dựa theo khoảng cách vật lý (Text nào nằm gần Detail nào nhất sẽ lấy tên của Detail đó).

4. DETAIL VIEW TOOLS

Hệ thống quản lý Bản vẽ xuất xưởng (A1 Frame) và Lịch sử cập nhật (Revision History).

  • Global Revision Bump: Tích vào đây nếu bạn muốn nâng hạng Revision (VD: từ REV.A lên REV.B) cho TOÀN BỘ các bản vẽ đang có trong danh sách cùng một lúc.
  • Nút [Sync & Update Sheet]: Quét các Block Khung A1 trên bản vẽ. Tool sẽ gom toàn bộ danh sách bản vẽ vào Bảng DataGrid phía dưới. Nó cũng hỗ trợ Auto-Heal (tự động phục hồi Text bị xóa) nếu kéo dữ liệu từ Excel về.
  • Nút [Create Rev Block]: Chọn một (hoặc nhiều) dòng trong bảng DataGrid, nhập nội dung Revision. Bấm nút này, Tool sẽ tự động xếp chồng (Stack) các Block Lịch sử sửa đổi ngay ngắn phía trên khung tên (Title Block) mà không bị đè lên nhau.
  • DataGrid (Sheets & Revs): Bảng điều khiển trung tâm. Bạn có thể Click chọn để tự động Pan/Zoom bản vẽ tới khung A1 tương ứng trên màn hình CAD.

Phụ lục: Hệ thống Ký hiệu Lực (Force Symbols)

Tool sử dụng tiêu chuẩn Hatch đặc (Solid) màu Trắng (khi in ra giấy sẽ là nét mực Đen đậm) để thể hiện các điểm tác dụng lực:

Loại Ký hiệu Hình dáng Ý nghĩa (Standard)
T1 ▲ Tam giác Vertical force (Sử dụng cho vị trí Deck, Stowed & Lifting). Đây là loại thông dụng và được gán mặc định.
T2 ■ Hình vuông Vertical force (Sử dụng cho vị trí Deck & Stowed).
T3 ● Hình tròn Vertical force (Chỉ sử dụng riêng cho vị trí Deck).
